## Steps

### 1. Build the Binary

First, compile the Aptos node binaries using Cargo:

```bash
cargo build
```

### 2. Start the Testnet

Run the Aptos testnet node in test mode:

```bash
mkdir -p local/test
cargo run -p aptos-node -- --test --test-dir ./local/test/
```

This command will generate the necessary configuration and logs. The output should look something like this:

```
WARNING: Entering test mode! This should never be used in production!
Completed generating configuration:
    Log file: "/path/to/aptos-core/local/test/validator.log"
    Test dir: "/path/to/aptos-core/local/test"
    Aptos root key path: "/path/to/aptos-core/local/test/mint.key"
    Waypoint: 0:1c31bdf97dc32b8d6f66d16e768945e2fd3ae5189b8cc87e301a3a73f398deb1
    ChainId: 4
    REST API endpoint: http://0.0.0.0:8080
    Metrics endpoint: http://0.0.0.0:9101/metrics
    Aptosnet fullnode network endpoint: /ip4/0.0.0.0/tcp/6181
    Admin service: http://0.0.0.0:9102/
```

### 3. Replace Key-File Path in Faucet Command

Now, you need to replace the `key-file-path` in the faucet service command with the root key path generated above. Run the following command to start the faucet service:

```bash
cargo run --package aptos-faucet-service -- run-simple --key-file-path "/path/to/aptos-core/local/test/mint.key" --node-url http://127.0.0.1:8080 --chain-id 4
```

### 4. Initialize Aptos CLI

You need to initialize the Aptos CLI. Run the following command to set up your configuration:

```bash
./target/debug/aptos init
```

This will generate the configuration file at `.aptos/config.yaml`, which contains account information.

### 5. Fund Your Account

To fund your account using the faucet, use the following command. Make sure to replace the `--account` with your actual account address (you can find it in the `config.yaml` file generated by the previous step):

```bash
./target/debug/aptos account fund-with-faucet \
  --account <YOUR_ACCOUNT_ADDRESS> \
  --amount 200000000 \
  --faucet-url http://127.0.0.1:8081 \
  --url http://127.0.0.1:8080
```

### 6. Run a Move Transaction

You can now interact with the Aptos blockchain by running a Move transaction. Replace `<YOUR_ACCOUNT_ADDRESS>` with your actual account address:

```bash
./target/debug/aptos move run \
  --function-id 0x1::add::debug_test_add \
  --max-gas 10000 \
  --gas-unit-price 1000 \
  --url http://0.0.0.0:8080 \
  --local \
  --sender-account <YOUR_ACCOUNT_ADDRESS>
```

This will execute the `debug_test_add` function on the testnet.
